# 🔐 Advanced Password Strength Checker

## 📌 Description
This project is an advanced password strength checker that evaluates how secure a password is based on multiple criteria such as length, uppercase and lowercase letters, numbers, and special characters.

It provides real-time feedback to help users create strong and secure passwords.

---

## 🚀 Features

- 🔍 Real-time password strength analysis  
- 📊 Strength indicator (Weak / Medium / Strong)  
- 📈 Visual strength bar  
- 💡 Suggestions for improving password  
- 🎨 Clean and user-friendly interface  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  

## 💡 How It Works

The password is evaluated based on the following rules:

- Minimum length (8+ characters)  
- Contains uppercase letters (A-Z)  
- Contains lowercase letters (a-z)  
- Includes numbers (0-9)  
- Includes special characters (!@#$ etc.)  

Based on these criteria, the password is classified as:

- Weak ❌  
- Medium ⚠️  
- Strong ✅
